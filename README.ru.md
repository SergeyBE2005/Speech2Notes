# Speech2Notes – Преобразование лекций в конспекты

Этот ноутбук преобразует аудиолекцию (на русском языке) в структурированный краткий конспект с использованием **Faster-Whisper** для транскрибации и **Groq (Llama 3.1)** для интеллектуального summarizing.

## Возможности

- Распознавание речи с помощью `faster-whisper` (модель large-v3, ускорение на CUDA)
- ИИ-суммаризация через Groq Llama 3.1 8B
- Результаты:
  - Полная транскрибация (`transcription.txt`)
  - Структурированный конспект (`summary.txt`) с:
    - Смысловыми блоками и подзаголовками
    - **Жирным** выделением ключевых терминов и определений
    - Маркированными списками для важных тезисов
    - Разделом "Главные выводы" в конце
- Работает в **Google Colab** (рекомендуется GPU)
- Сохраняет результаты в Google Drive и скачивает их локально

## Как это работает

1. Монтирует Google Drive для доступа к аудиофайлу.
2. Устанавливает необходимые пакеты (`groq`, `faster-whisper`).
3. Транскрибирует аудио с помощью Faster-Whisper (русский язык).
4. Отправляет транскрибацию в LLM Groq с специально разработанным промптом.
5. Сохраняет и скачивает как исходную транскрибацию, так и итоговый конспект.

## Требования

- Google Colab с **GPU-ускорителем** (T4 или лучше)
- Аудиофайл в поддерживаемом формате (в примере используется `.m4a`)
- API-ключ Groq (доступен бесплатный тариф)

## Настройка

1. Загрузите вашу аудиолекцию в Google Drive (например, `Colab Notebooks/Notes-maker/Data/`).
2. Обновите переменную `AUDIO_PATH` в ноутбуке.
3. Замените `GROQ_API_KEY` на ваш собственный ключ.
4. Запустите все ячейки.
